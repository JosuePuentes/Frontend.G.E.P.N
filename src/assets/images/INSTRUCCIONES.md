# Instrucciones para Agregar Imágenes

## Imágenes Requeridas

Para que el HomeScreen se muestre correctamente, necesitas agregar las siguientes imágenes en esta carpeta:

### 1. bandera-venezuela.png
- **Descripción**: Bandera de Venezuela
- **Tamaño recomendado**: 300x200px o proporción 3:2
- **Formato**: PNG con fondo transparente o JPG

### 2. policias-tacticos.png
- **Descripción**: Imagen de policías tácticos
- **Tamaño recomendado**: 300x300px (cuadrada)
- **Formato**: PNG o JPG

### 3. patrullas.png
- **Descripción**: Imagen de patrullas policiales
- **Tamaño recomendado**: 300x300px (cuadrada)
- **Formato**: PNG o JPG

## Cómo Agregar las Imágenes

1. Descarga o crea las imágenes con los nombres exactos mencionados arriba
2. Colócalas en esta carpeta: `src/assets/images/`
3. Asegúrate de que los nombres de archivo sean exactamente:
   - `bandera-venezuela.png`
   - `policias-tacticos.png`
   - `patrullas.png`

## Nota

Si las imágenes no están disponibles, la aplicación seguirá funcionando pero no se mostrarán las imágenes. El título y el botón de login se mostrarán normalmente.
